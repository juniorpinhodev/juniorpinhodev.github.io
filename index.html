Sobre Junior Pinho

Desenvolvedor FullStack

Olá! Sou um desenvolvedor apaixonado pela tecnologia e em constante busca pelo próximo desafio. Atualmente, estou cursando graduação em Ciência da Computação no Centro Universitário União das Américas Descomplica, o que me proporciona uma base sólida para meu trabalho.

Minha jornada na programação começou com as bases: linguagens Java e C (lógica de programação, algoritmos e programação orientada a objetos) e rapidamente expandiu-se para incluir uma ampla gama de tecnologias, como HTML, CSS e JavaScript, C#, Python, SQL e linguagem R.
Além disso, estou bem familiarizado com ferramentas de controle de versão, como Git e GitHub.

Meu portfólio inclui uma série de projetos interessantes, que vão desde web sites responsivos criados em Javascript; análises de dados em SQL e Python; até aplicativos, robôs e automações criados com Python e C#. Cada projeto é uma oportunidade para aplicar minha paixão pela resolução de problemas e minha criatividade na criação de soluções práticas e eficientes. “Vamos desenvolver algo incrível juntos!”.
